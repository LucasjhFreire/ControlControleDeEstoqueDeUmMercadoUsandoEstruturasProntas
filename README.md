🛒 Controle de Estoque com ArrayList (Java)
Sistema de gerenciamento de estoque para supermercados desenvolvido em Java para fins acadêmicos. O objetivo principal do projeto foi aplicar o uso de estruturas de dados prontas da linguagem para criar um código limpo, eficiente e moderno.

⚡ Como o sistema funciona?
Em vez de vetores fixos, o estoque utiliza a API de coleções do Java para manipulação dinâmica dos dados em memória:

Armazenamento Dinâmico: Uso de ArrayList para permitir que o estoque cresça sem limite rígido de tamanho e para facilitar a remoção de itens sem precisar reorganizar índices manualmente.

Ordenação Eficiente: Utilização do método Collections.sort() junto com a interface Comparator para ordenar os produtos por nome de forma rápida e otimizada.

🛠️ Funcionalidades
➕ Cadastrar produto (armazenamento dinâmico em lista)

❌ Remover produto (exclusão nativa por índice ou objeto)

🔄 Atualizar preço do produto

🔍 Pesquisar produto por código

📋 Listar estoque completo

🔤 Listar produtos ordenados por nome (Collections + Comparator)
